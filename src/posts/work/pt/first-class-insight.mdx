## Visão Geral

**First Class Insight** é uma plataforma SaaS de conteúdo executivo. Reúne resumos, cursos e estudos de caso pensados para quem tem pouco tempo e precisa de informação densa, aplicável e direta ao ponto.

## Problema

Líderes e empreendedores são bombardeados por conteúdo, mas falta curadoria. Ler um livro inteiro ou um curso de horas nem sempre cabe na agenda — e o conhecimento que importa fica adiado.

## Solução

Conteúdo executivo organizado para consumo rápido:

- **Resumos** de livros e temas de negócio em formato direto
- **Cursos** curtos e aplicáveis, por trilha
- **Estudos de caso** reais com aprendizados acionáveis
- **Biblioteca** com busca e organização por tema
- **Acompanhamento** de progresso e favoritos

## Stack Técnica

- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Node.js + API REST
- **Banco**: PostgreSQL para catálogo, usuários e progresso
- **Infra**: deploy em nuvem com CDN para mídia

## Impacto

O First Class Insight transforma tempo morto em aprendizado. Em vez de acumular conteúdo para "ler depois", o usuário consome o essencial e aplica no negócio.
