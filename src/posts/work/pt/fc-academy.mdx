## Visão Geral

**FC Academy** é uma escola de tecnologia online focada em projetos reais. Em vez de aulas soltas e teoria isolada, o aluno aprende construindo — do primeiro commit ao deploy.

## Problema

Muito curso de programação ensina sintaxe, mas não ensina a entregar. O aluno termina sabendo "fazer um exercício", mas trava diante de um projeto de verdade, com requisitos, prazo e código de outras pessoas.

## Solução

Formação prática, por trilha e por projeto:

- **Trilhas** de aprendizado do básico ao avançado
- **Projetos reais** que viram portfólio
- **Aulas** em vídeo com material de apoio
- **Acompanhamento** de progresso e entregas
- **Comunidade** para dúvidas e code review

## Stack Técnica

- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Node.js + API REST
- **Banco**: PostgreSQL para cursos, matrículas e progresso
- **Mídia**: streaming de vídeo via CDN

## Impacto

A FC Academy encurta a distância entre "estudar" e "trabalhar com isso". O aluno sai com projetos no portfólio e com a prática que o mercado realmente cobra.
