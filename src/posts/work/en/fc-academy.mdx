## Overview

**FC Academy** is an online tech school focused on real projects. Instead of loose lessons and isolated theory, students learn by building — from the first commit to deploy.

## Problem

Many programming courses teach syntax but not delivery. Students finish knowing how to "do an exercise" but freeze in front of a real project, with requirements, deadlines and other people's code.

## Solution

Hands-on training, by track and by project:

- **Learning tracks** from basics to advanced
- **Real projects** that become a portfolio
- **Video lessons** with supporting material
- **Tracking** of progress and deliverables
- **Community** for questions and code review

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Node.js + REST API
- **Database**: PostgreSQL for courses, enrollments and progress
- **Media**: video streaming via CDN

## Impact

FC Academy shortens the distance between "studying" and "working with it". Students leave with portfolio projects and the practice the market actually demands.
