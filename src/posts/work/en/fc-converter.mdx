## Overview

**FC Converter** is a cloud file converter with no install and no usage limit. The user uploads a file, picks the output format and downloads the result — all in the browser.

_"Unlimited file converter."_

## Problem

Online converters are usually full of ads, with daily file limits and doubts about what happens to the data you upload. Anyone converting in volume ends up stuck with paid or unsafe software.

## Solution

Fast, unlimited conversion right in the browser:

- **Documents**: PDF, DOCX, XLSX, PPTX and more
- **Images**: PNG, JPG, WEBP, SVG
- **Audio and video**: MP3, MP4, WAV and others
- **Batch**: multiple files at once
- **Privacy**: files processed and discarded after conversion

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Node.js with queued conversion workers
- **Processing**: server-side conversion tools (e.g. FFmpeg for media)
- **Infra**: temporary cloud storage with automatic expiration

## Impact

FC Converter solves file conversion without friction: nothing to install, no artificial limits, and no files left sitting on a server. Handy for anyone dealing with many formats daily.
