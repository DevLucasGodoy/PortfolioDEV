## Overview

**First Class Insight** is a SaaS executive content platform. It brings together summaries, courses and case studies designed for people with little time who need dense, applicable, to-the-point information.

## Problem

Leaders and entrepreneurs are bombarded with content but lack curation. Reading a whole book or a multi-hour course doesn't always fit the agenda — and the knowledge that matters keeps getting postponed.

## Solution

Executive content organized for fast consumption:

- **Summaries** of books and business topics in a direct format
- **Short, applicable courses** organized by track
- **Real case studies** with actionable takeaways
- **Library** with search and topic organization
- **Tracking** of progress and favorites

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Node.js + REST API
- **Database**: PostgreSQL for catalog, users and progress
- **Infra**: cloud deploy with CDN for media

## Impact

First Class Insight turns dead time into learning. Instead of piling up content to "read later", the user consumes the essentials and applies them to the business.
