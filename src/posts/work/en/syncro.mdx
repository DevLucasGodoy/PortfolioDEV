## Overview

**Syncro** is a SaaS integration and automation platform that connects different systems without writing code. Instead of spreadsheets copied by hand between tools, companies build flows that sync data automatically.

_"Your systems and your team in sync."_

## Problem

SMBs use dozens of tools that don't talk to each other — ERP, spreadsheets, e-commerce, CRM. Integration becomes manual, error-prone work, and the data is never up to date everywhere at once.

## Solution

A visual automation builder linking systems by trigger and action:

- **Connectors** for ERPs, spreadsheets, APIs and external services
- **Flows** built visually (trigger → condition → action), no code
- **Synchronization** scheduled or in real time
- **Logs** and reprocessing of failed executions
- **Field mapping** between source and destination systems

## Tech Stack

- **Frontend**: React, TypeScript — visual flow editor
- **Backend**: Node.js, execution engine with queues and workers
- **Database**: PostgreSQL for flow definitions and execution history
- **Integrations**: REST, webhooks and custom connectors

## Impact

Syncro ends double data entry between systems and keeps data consistent across the whole operation. The team stops being a "manual bridge" between tools and focuses on what matters.
