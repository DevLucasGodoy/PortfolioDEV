## Overview

**FC Chef** is a SaaS recipe platform that connects what you want to cook to the shopping list and the weekly menu. Instead of deciding last minute, the user plans meals and the app handles the rest.

_"Master Chef."_

## Problem

The question "what am I going to cook today?" repeats every day. The result is unplanned grocery shopping, ingredients spoiling in the fridge and wasted time and money.

## Solution

From planning to the store, in one flow:

- **Recipes** with ingredients, instructions and servings
- **Weekly menu** built by drag and drop
- **Shopping list** generated automatically from the menu
- **Favorites** and search by ingredient or category
- **Serving adjustment** that recalculates the shopping list

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS (responsive / PWA)
- **Backend**: Node.js + REST API
- **Database**: PostgreSQL for recipes, menus and lists
- **Infra**: cloud deploy

## Impact

FC Chef ends daily indecision and ingredient waste. Planning the week takes minutes, and the trip to the store no longer holds surprises.
