## Overview

**FC Noivos** is a SaaS platform for end-to-end wedding planning. It brings together guest list, budget, vendors and checklist in one place, so the couple can organize the big day without losing track.

_"Your big day, planned with love."_

## Problem

Planning a wedding means managing dozens of fronts at once — guests, tables, vendors, budget, deadlines. It usually lives scattered across spreadsheets and chats, and any forgotten detail is costly.

## Solution

The whole wedding organized, from engagement to the altar:

- **Guests** with RSVP, plus-ones and table arrangement
- **Budget** with spend tracking versus plan
- **Vendors** with contracts, payments and contacts
- **Checklist** with tasks and deadlines by stage
- **Timeline** from preparation to the event day

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS (responsive)
- **Backend**: Node.js + REST API
- **Database**: PostgreSQL with data isolated per couple/event
- **Infra**: cloud deploy

## Impact

FC Noivos turns planning chaos into a clear, calm dashboard. The couple tracks everything in one place and reaches the big day feeling in control — not in a rush.
