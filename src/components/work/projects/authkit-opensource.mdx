## Visão Geral

**AuthKit** é uma biblioteca para Node.js que resolve autenticação de forma completa e sem vendor lock-in. O objetivo foi criar uma solução que fosse simples de integrar mas flexível o suficiente para casos de uso avançados.

## Motivação

As soluções de autenticação existentes ou eram muito simples (só JWT) ou muito complexas e caras (Auth0, Clerk). O AuthKit preenche esse meio-termo: completo, gratuito e sob controle total do desenvolvedor.

## Funcionalidades

- **JWT + Refresh Tokens** com rotação automática
- **OAuth2** com suporte a Google, GitHub e qualquer provider compatível
- **Magic Links** por e-mail
- **Passkeys** (WebAuthn) para autenticação sem senha
- **Rate limiting** integrado por padrão
- **Logs de auditoria** para cada evento de autenticação

## Exemplo de uso

```ts
import { AuthKit } from "authkit";

const auth = new AuthKit({
  secret: process.env.AUTH_SECRET,
  providers: ["google", "github"],
});

app.post("/login", auth.handleLogin());
app.post("/refresh", auth.handleRefresh());
```
